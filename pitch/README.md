# Aurora Travels pitch deck

Generate the investor deck (populated with cited Kenya tourism / M-Pesa / market figures):

```bash
npm install
npm run build:deck
```

Output: `pitch/AuroraTravels_Pitch_Deck.pptx`

Sources used in the deck include Kenya Ministry of Tourism & Wildlife / Magical Kenya (2025 sector performance), WTTC EIR, Communications Authority of Kenya, Federation of Kenya Employers, and OTA commission benchmarks.
